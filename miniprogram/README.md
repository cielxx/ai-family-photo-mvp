# 微信小程序 mock 版

这是“AI 家庭纪念影像助手”的微信小程序本地 mock 版，用于先验证移动端主流程。

当前版本暂不接入云开发、Dify Workflow 或真实图像 API。页面中的处理结果来自 `assets/mock/` 测试图片。

## 如何打开

使用微信开发者工具打开当前目录：

```text
miniprogram/
```

如果还没有正式小程序 AppID，可以先用测试号或游客模式打开。当前 `project.config.json` 使用的是 `touristappid` 占位。

## 当前流程

1. 首页
2. 服务选择
3. 上传照片或使用示例照片
4. 选择处理重点并确认授权
5. 生成中 mock 状态
6. 初次结果展示
7. 二次调整
8. 调整后结果展示
