# PROJECT SUMMARY | AI 家庭纪念影像助手 MVP

## 1. 项目总结

围绕家庭纪念影像场景，设计并搭建一个 AI 影像处理 MVP，通过小程序、云开发与 Dify Workflow，跑通从用户提交任务、AI 理解需求、结果确认到二次调整的端到端闭环。

## 2. 项目目标

本项目的目标是验证家庭纪念影像处理服务的核心链路：降低用户表达图像处理需求的门槛，将自然语言需求转化为可执行任务，并为后续接入真实图像处理 API 保留清晰的数据与流程接口。

## 3. 核心产品判断

### 3.1 不做大而全的修图工具

家庭纪念影像用户通常缺少专业修图经验，产品重点放在低门槛、可理解、可调整的影像处理服务。

### 3.2 优先跑通核心闭环

项目范围聚焦核心处理链路，真实图像处理 API、登录、订单、支付等完整业务模块作为后续扩展项。当前链路覆盖：

用户需求输入 → AI 任务生成 → 图像处理结果回传 → 二次调整

### 3.3 AI Workflow 是需求转译层

本项目中的 AI Workflow 用于将用户自然语言需求转化为可执行任务卡和图像处理 Prompt。
