# PROJECT SUMMARY | AI 家庭纪念影像助手 MVP

## 1. 项目总结

围绕家庭纪念影像场景，设计并搭建一个 AI 影像处理 MVP，通过 Streamlit + Dify Workflow + 半自动图像处理方案，跑通从用户输入、AI 理解、图像处理结果展示到二次调整的端到端闭环。

## 2. 项目目标

本项目的目标不是立即交付完整商业化产品，而是完成一个可运行、可演示、可写入作品集的 AI 产品 MVP，用于展示从需求定义、原型设计、AI Workflow 设计、Demo 搭建到测试复盘的完整过程。

## 3. 核心产品判断

### 3.1 不做大而全的修图工具

家庭纪念影像用户通常不是专业修图用户，因此产品重点不是提供复杂编辑能力，而是提供低门槛、可理解、可调整的影像处理服务。

### 3.2 v0.1 先验证闭环

当前版本没有接入真实图像处理 API，也没有加入登录、订单、支付等商业功能。v0.1 的核心目标是验证：

用户需求输入 → AI 任务生成 → 图像处理结果回传 → 二次调整

### 3.3 AI Workflow 是需求转译层

本项目中的 AI Workflow 不是简单聊天功能，而是将用户自然语言需求转化为可执行任务卡和图像处理 Prompt 的中间层。
