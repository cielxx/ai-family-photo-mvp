# 项目立项卡 v0.1

## 项目名称

AI 家庭纪念影像助手

## 项目定位

一个面向家庭用户的 AI 影像处理 MVP，帮助用户完成老照片修复、黑白照片上色和家庭纪念照生成。

## 目标用户

1. 有老照片修复需求的中老年用户
2. 为父母或长辈处理照片的子女用户
3. 有婚纱照、全家福、节日纪念图等家庭影像需求的用户

## 核心痛点

用户希望修复旧照片或生成具有纪念意义的家庭影像，但不会使用复杂修图软件，也不了解 AI 图像工具的 prompt 写法和操作流程。

## MVP 功能范围

1. 上传照片
2. 选择服务类型
3. 填写补充需求
4. 生成或上传处理结果
5. 展示前后对比
6. 收集满意度和修改意见

## 第一版服务类型

1. 老照片修复
2. 黑白照片上色
3. 婚纱照 / 纪念照生成

## 暂不做

1. 支付功能
2. 用户登录
3. 小程序上线
4. 面向陌生用户公开上传
5. 多人全家福稳定生成
6. 批量处理
7. 直播接单商业化

## 技术实现策略

不将底层能力绑定于单一模型服务，而采用“可替换图像模型接入层”的设计思路。

v0.1 阶段优先实现产品主流程：

用户上传照片 → 选择服务类型 → 系统生成结构化任务参数与 Prompt → 图像生成/编辑 → 结果展示 → 用户反馈记录。

图像生成与编辑服务将在开发阶段具体选型，优先评估 OpenAI API、国内图像生成/编辑 API 或其他可用模型服务。若 API 暂不可用，则采用半自动 MVP：系统生成 Prompt 与任务卡，由人工调用可用 AI 图像工具完成生成，再回传结果进行展示与测试。

## 合规边界

项目初期仅使用本人、家人或明确授权样本进行测试，暂不面向公众开放真人照片上传服务。

用户上传真人照片涉及肖像权、隐私和个人信息保护问题，后续若进入公开测试或商业化阶段，需要补充用户授权、隐私政策、数据删除机制和生成内容标识等设计。
