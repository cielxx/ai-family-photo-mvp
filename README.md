# AI 家庭纪念影像助手

本项目是一个面向家庭用户的 AI 影像处理 MVP，聚焦老照片修复、黑白照片上色与家庭纪念照生成场景。

## 项目目标

验证 AI 图像生成与编辑能力在家庭纪念影像场景中的应用可行性，并完成从用户需求分析、产品流程设计、原型设计、MVP 搭建到测试评估的 AI 产品闭环。

## 当前阶段

Day 1：项目立项与仓库初始化。

## 初步功能范围

- 上传照片
- 选择服务类型
- 填写补充需求
- 生成或上传处理结果
- 展示前后对比
- 收集用户反馈

## 技术路线

本项目采用“可替换模型服务”的设计思路。第一阶段计划使用 Python + Streamlit 搭建 Web Demo。

图像生成与编辑能力将优先评估 OpenAI API、国内图像生成/编辑 API 或其他可用模型服务。若 API 暂不可用，则先采用半自动 MVP 流程验证产品路径。
